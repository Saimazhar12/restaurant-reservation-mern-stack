# 🍽️ Restaurant Reservation System (MERN Stack)

A full-stack web application built using MongoDB, Express.js, React, and Node.js that allows users to easily book and manage restaurant reservations through a simple and responsive interface.

# 📌 Project Overview
The goal of this project is to develop a complete restaurant reservation system where users can submit booking requests and store them in a database. This project demonstrates full-stack development skills including frontend design, backend API development, and database integration.

# 📂 Dataset / Data Handling
- Stores user reservation data (name, email, date, time)
- Data is saved in MongoDB
- Real-time form submission and storage

# ⚙️ Features
- 📝 Create restaurant reservations  
- 📦 Store reservation data in MongoDB  
- 🔗 RESTful API integration  
- ⚡ Fast and responsive UI  
- 📁 Structured backend architecture  
- 🌐 Full MERN stack implementation  

# 🧠 Backend
- Node.js with Express.js  
- MongoDB database  
- Mongoose schemas/models

- Error handling  

# 💻 Frontend
- React.js user interface

# 🚀 How to Run

## 1. Clone Repository
Clone this repository to run and explore the web application locally.

## 2. Setup Backend
cd Backend  
npm install  
npm run dev   

## 3. Setup Frontend
cd Frontend  
npm install  
npm run dev  

## 4. Environment Variables
Create a `.env` file in Backend folder:

PORT=4000  
MONGO_URI=your_mongodb_connection_string  

# 📊 Usage
- Open the frontend in browser  
- Fill reservation form  
- Submit booking  
- Data will be saved in MongoDB
